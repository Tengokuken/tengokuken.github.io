My goals

I've always wanted to be able to have a positive impact on others. Whether that be in the form educating, creating cool things that people can use, or just making others laugh. I've felt that coding was one avenue to express this goal.

In my career, I'd like to learn and master good industry coding styles. I would like to create code that is not only efficient, but is readable and easily maintainable by others who would use it down the line.

In addition, I would love to learn all different kinds of technologies that are used in the industry. The CS field is growing rapidly, and learning how to do new things and different methods of doing the same thing satiates my desire to learn!

I would also like to improve my communication skills. I believe that communication can easily make or break the end result of a project that a team is working on. If everyone is on the same page, then the product will look exactly as the team envisioned.

I've been programming for around 6 years and even now continue to strive to improve.
